A Máquina de Turing como G-Chain · The Turing Machine as G-Chain
🇧🇷 Português — Primário

Uma máquina de Turing é uma cadeia de operações: ler o símbolo (C — comprimir estado), consultar a tabela de transição (K — constranger ao próximo estado), escrever e mover (F — dobrar o meio), atualizar o estado interno (U — desdobrar para a próxima configuração). A estrutura G = U∘F∘K∘C não é uma metáfora da máquina de Turing — é o mesmo objeto descrito em linguagem geométrica.

Teorema de Turing 1936
Não existe máquina de Turing M que, dado qualquer par (P, x), decide em tempo finito se P(x) para. Equivalentemente: o conjunto $\{(P,x) : P(x)\text{ para}\}$ não é decidível.

No dm³, a indecidibilidade aparece quando o operador K tenta constranger a cadeia G a uma parada garantida. O teorema de Turing afirma que essa restrição não pode ser imposta uniformemente sobre todo o espaço de entradas.

🇺🇸 English

A Turing machine is a chain of operations: read the symbol (C — compress state), consult the transition table (K — constrain to next state), write and move (F — fold the tape), update internal state (U — unfold to next configuration). G = U∘F∘K∘C is not a metaphor for the Turing machine — it is the same object described in geometric language.

Turing's Theorem 1936
No Turing machine M can, given any pair (P, x), decide in finite time whether P(x) halts. Equivalently: the set $\{(P,x): P(x)\text{ halts}\}$ is not decidable.

In dm³, undecidability appears when operator K attempts to constrain the G-chain to a guaranteed halt. Turing's theorem states this constraint cannot be imposed uniformly over all inputs.

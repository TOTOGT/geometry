John Hopfield
Memória como Ponto Fixo do G-Chain

Em 1982, Hopfield mostrou que uma rede neural com pesos simétricos é um sistema físico com energia decrescente. Cada memória armazenada é um atrator — um ponto fixo. Em dm³, isso é G(x*) = x*.

In 1982, Hopfield showed a neural network with symmetric weights is a physical system with decreasing energy. Each stored memory is an attractor — a fixed point. In dm³ this is G(x*) = x*.

O Hamiltoniano de Hopfield como Operador F · The Hopfield Hamiltonian as Operator F
', '
🇧🇷 Português

A rede de Hopfield armazena padrões $\\{\\xi^\\mu\\}$ como pesos $w_{ij} = \\frac{1}{N}\\sum_\\mu \\xi^\\mu_i \\xi^\\mu_j$. A dinâmica minimiza a energia $E(s) = -\\frac{1}{2}\\sum_{ij} w_{ij} s_i s_j$. O sistema é garantido a convergir para um mínimo local — um padrão armazenado ou uma mistura espúria.

Hopfield-dm³
A energia de Hopfield $E(s) = -\\frac{1}{2}s^T W s$ é o potencial de Lyapunov do operador F. Cada mínimo de $E$ é um ponto fixo $x^*$ do G-chain com $F(x^*) = x^*$. A curvatura do poço é $\\kappa \\leq \\sqrt{7/9}$ (Teorema da Recursão, GTCT Vol IV Ch 6). O expoente de Lyapunov é $\\mu_{\\max} = -2$.

A aprendizagem Hebbiana — aumentar $w_{ij}$ quando $s_i$ e $s_j$ são co-ativos — é o operador C: comprimir a correlação de atividade para um peso sináptico escalar.

🇺🇸 English

The Hopfield network stores patterns $\\{\\xi^\\mu\\}$ as weights $w_{ij} = \\frac{1}{N}\\sum_\\mu \\xi^\\mu_i \\xi^\\mu_j$. Dynamics minimise energy $E(s) = -\\frac{1}{2}\\sum_{ij} w_{ij} s_i s_j$. The system is guaranteed to converge to a local minimum — a stored pattern or a spurious mixture.

Hopfield–dm³
Hopfield energy $E(s) = -\\frac{1}{2}s^T W s$ is the Lyapunov potential of operator F. Each minimum of $E$ is a fixed point $x^*$ of the G-chain with $F(x^*) = x^*$. The curvature of the well is $\\kappa \\leq \\sqrt{7/9}$ (Recursion Theorem, GTCT Vol IV Ch 6). The Lyapunov exponent is $\\mu_{\\max} = -2$.

Hebbian learning — increasing $w_{ij}$ when $s_i$ and $s_j$ are co-active — is operator C: compressing activity correlation into a scalar synaptic weight.

g₃₃ = 33 e a Capacidade da Rede · g₃₃ = 33 and Network Capacity
', '
🇧🇷 Português

A capacidade da rede de Hopfield é aproximadamente $0.14N$ padrões para $N$ neurônios. Para $N = 33$ (g₃₃), a capacidade é aproximadamente 4–5 padrões sem erro. Isso não é coincidência: g₃₃ = 33 é o número de ciclos necessários para que o G-chain atinja o limiar de estabilidade cortical (Teorema da Ortogonalidade, GTCT).

g₃₃ e Capacidade
33 neurônios = 4–5 memórias estáveis = um ciclo completo de compressão-dobramento-constrangimento-desdobramento. O praticante precisa de 33 repetições não porque o corpo é lento, mas porque o G-chain precisa de 33 iterações para convergir ao ponto fixo $x^*$ com $\\kappa < 1$.

🇺🇸 English

The Hopfield network\'s capacity is approximately $0.14N$ patterns for $N$ neurons. For $N = 33$ (g₃₃), capacity is roughly 4–5 patterns without error. This is not coincidence: g₃₃ = 33 is the number of cycles required for the G-chain to reach the cortical stability threshold (Orthogonality Theorem, GTCT).

g₃₃ and Capacity
33 neurons = 4–5 stable memories = one complete compress-fold-constrain-unfold cycle. The practitioner needs 33 repetitions not because the body is slow, but because the G-chain needs 33 iterations to converge to fixed point $x^*$ with $\\kappa < 1$.
